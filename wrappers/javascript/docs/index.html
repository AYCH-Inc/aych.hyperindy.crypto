Indy Crypto API Reference¶

indycrypto.blsGenerator()¶

Creates and returns random generator point that satisfies BLS algorithm requirements.

Returns:A Generator.
indycrypto.blsGeneratorAsBytes(generator)¶

Returns BLS generator point bytes representation.

Arguments:
  • generator – The generator.
Returns:

A byte array.
indycrypto.blsGeneratorFromBytes(bytes)¶

Creates and returns generator point from bytes representation.

Arguments:
  • bytes – An array of bytes, as returned by blsGeneratorAsBytes.
Returns:

A Generator.
indycrypto.blsSignKey([seed])¶

Creates and returns random (with optional seed) BLS sign key algorithm requirements.

Arguments:
  • seed – A seed as an array of bytes (length 32).
Returns:

A SignKey.
indycrypto.blsSignKeyAsBytes(signKey)¶

Returns BLS sign key bytes representation.

Arguments:
  • signKey – A SignKey.
Returns:

A byte array.
indycrypto.blsSignKeyFromBytes(bytes)¶

Creates and returns BLS sign key from bytes representation.

Arguments:
  • bytes – An array of bytes, as returned by blsSignKeyAsBytes.
Returns:

A SignKey.
indycrypto.blsVerKey(generator, signKey)¶

Creates and returns BLS ver key that corresponds to sign key.

Arguments:
  • generator – A Generator.
  • signKey – A SignKey.
Returns:

A VerKey.
indycrypto.blsVerKeyAsBytes(verKey)¶

Returns BLS verification key bytes representation.

Arguments:
  • verKey – A VerKey.
Returns:

A byte array.
indycrypto.blsVerKeyFromBytes(bytes)¶

Creates and returns BLS verification key from bytes representation.

Arguments:
  • bytes – An array of bytes, as returned by blsVerKeyAsBytes.
Returns:

A VerKey.
indycrypto.blsProofOfPossession(verKey, signKey)¶

Creates and returns BLS proof of possession that corresponds to ver key.

Arguments:
  • verKey – A VerKey.
  • signKey – A SignKey.
Returns:

A ProofOfPossession.
indycrypto.blsProofOfPossessionAsBytes(proofOfPossession)¶

Returns BLS proof of possession to bytes representation.

Arguments:
  • proofOfPossession – A ProofOfPossession.
Returns:

A byte array.
indycrypto.blsProofOfPossessionFromBytes(bytes)¶

Creates and returns BLS proof of possession from bytes representation.

Arguments:
  • bytes – An array of bytes, as returned by blsProofOfPossessionAsBytes.

indycrypto.blsMultiSignature(signatures)¶

  • signatures – An array of signatures.

indycrypto.blsSign(message, signKey)¶

Signs the message and returns signature.

Arguments:
  • message (string) – The message to sign.
  • signKey – A SignKey.
Returns:

A Signature.
indycrypto.blsVerify(signature, message, verKey, generator)¶

Verifies the message signature and returns true if signature valid or false otherwise.

Arguments:
  • signature – Signature to verify.
  • message (string) – Message to verify.
  • verKey – A VerKey.
  • generator – A Generator.
Returns:

boolean.
indycrypto.blsVerifyProofOfPossession(proofOfPossession, verKey, generator)¶

Verifies the proof of possession and returns true if signature valid or false otherwise.

Arguments:
  • proofOfPossession – Proof of possession.
  • verKey – A VerKey.
  • generator – A Generator.
Returns:

boolean.
indycrypto.blsVerifyMultiSig(multiSig, message, verKeys, generator)¶

Verifies the message multi signature and returns true if signature valid or false otherwise.

Arguments:
  • multiSig – MultiSignature to verify.
  • message (string) – Message to verify.
  • verKeys – An array of VerKeys.
  • generator – A Generator.

boolean.
